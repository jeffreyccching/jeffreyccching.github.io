Publications

Declarative Demand-Driven Reverse Engineering.
Binary reverse engineering is a challenging task because it often necessitates reasoning using both domain-specific knowledge and logical inference. To help perform these tasks, reverse engineers often use toolkits that allow them to interactively explicate properties of binaries. We argue that deductive databases serve as a natural abstraction for interfacing between visualization-based binary analysis tools and high-performance logical inference engines that compute facts about binaries.
Yihao Sun, Jeffrey Ching, Kristopher Micinski
